2. 结构说明

• 顶部：两个旋转的"旋桨"字符

• 机身：两行"直-升-机"字符构成主体

• 底部：

  • 左侧：机枪发射口

  • 右侧：导弹发射器

  • 中间：漏油的油箱

• 尺寸：宽7格，高3行的大型BOSS

• 颜色：军绿色机身，红色警示灯

二、血量阶段划分

阶段划分

• 阶段A：100-71 HP（前30血）- 激光阶段

• 阶段B：70-41 HP（中间30血）- 导弹阶段

• 阶段C：40-11 HP（后30血）- 冲撞阶段

• 阶段D：10-0 HP（最后10血）- 混合攻击阶段

三、攻击能力详细说明

阶段A：激光攻击（100-71 HP）

攻击模式：
• 攻击方式：从底部发射红色激光

• 激光字符："光"字，红色

• 攻击频率：每5秒一次

• 持续时间：2秒激光束

• 攻击范围：垂直向下的激光柱

• 伤害：每秒10点伤害

特殊机制：
• 激光只对非商鞅形态造成伤害

• 商鞅形态站在激光中不会受伤

• 直升机在发射激光时悬停不动

玩家应对：
• 必须切换商鞅形态

• 用商鞅的"头部"向上攻击直升机

• 用商鞅的"手部"抓取激光反射

• 反击成功造成5-10点伤害

阶段B：导弹攻击（70-41 HP）

攻击模式：
• 攻击方式：发射抛物线追踪导弹

• 导弹字符："导"字，有尾焰

• 发射数量：每次2-3枚

• 发射频率：每4秒一次

• 导弹速度：中等

• 爆炸范围：2×2格

• 单发伤害：15点

特殊机制：
• 导弹只能用黑曼巴的"肘击"弹反

• 弹反后的导弹会自动追踪直升机

• 导弹可被提前击落

玩家应对：
• 必须切换黑曼巴形态

• 用肘击在导弹接近时弹反

• 弹反导弹命中直升机造成20点伤害

• 可用篮球提前击落导弹

阶段C：冲撞攻击（40-11 HP）

攻击模式：
• 攻击方式：直线高速冲撞

• 冲撞准备：后退到屏幕边缘，蓄力2秒

• 冲撞速度：极快

• 冲撞宽度：3格高度范围

• 伤害：被撞即死

• 眩晕：撞墙后眩晕3秒

• 攻击频率：每6秒一次

特殊机制：
• 只能用哈基米的"弓背形态"跳跃躲避

• 跳跃高度需足够跳过直升机

• 冲撞路径固定直线

玩家应对：
• 必须切换哈基米形态

• 使用弓背形态（按Q切换"基"）

• 在冲撞瞬间跳跃

• 在空中用哈气攻击直升机

• 哈气命中造成8点伤害

阶段D：混合攻击（10-0 HP）

攻击模式：
• 攻击顺序：激光 → 导弹 → 冲撞 → 重复

• 攻击间隔：初始2秒，随血量减少而加快

• 攻击频率：越来越快，最后几乎无间隔

• 攻击循环：完整执行三连击

特殊机制：
• 必须按顺序切换对应形态

• 一次失误扣除20点生命

• 连续正确应对可获得额外伤害加成

• 最后1血时攻击速度达到极限

玩家应对：
• 快速在三种形态间切换

• 记住攻击顺序：激光→导弹→冲撞

• 每个攻击前有1秒预警时间

• 完美应对可获得短暂输出窗口

四、移动模式

阶段相关移动

阶段A移动：
• 悬停在玩家正上方5格高度

• 轻微左右移动保持对准

• 发射激光时完全静止

阶段B移动：
• 在屏幕顶端左右移动

• 改变导弹发射角度

• 躲避玩家反弹的导弹

阶段C移动：
• 从屏幕一侧边缘开始

• 直线冲撞到另一侧

• 撞墙后返回起始位置

阶段D移动：
• 根据当前攻击类型移动

• 快速变换位置

• 移动轨迹难以预测

特殊移动

被击中反馈：
• 头部击中：螺旋桨停转0.5秒

• 导弹反弹：机身倾斜，短暂失控

• 哈气命中：后退并轻微摇晃

濒死表现：
• HP<5：移动速度减慢30%

• HP<3：螺旋桨转速不稳定

• HP=1：摇摇欲坠，有下坠趋势

五、视觉与音效提示

血量显示


分段血条：

• 导弹转向，尾焰变色

八、难度与平衡

伤害数值

玩家攻击：
• 商鞅头部：5点/次

• 导弹弹反：20点/次

• 哈基米哈气：8点/次

• 其他攻击：2-3点/次

直升机攻击：
• 激光：10点/秒

• 导弹：15点/发

• 冲撞：即死

• 最终阶段失误：20点/次

时间压力

建议战斗时长：3-4分钟
阶段时间限制：
• 阶段A：1分钟内完成

• 阶段B：1分钟内完成

• 阶段C：1分钟内完成

• 阶段D：30秒内完成

超时惩罚：
• 攻击频率加快

• 预警时间缩短

• 伤害增加

容错设计

新手保护：
• 前两次失误不扣血

• 预警时间延长0.5秒

• 攻击速度降低20%

熟练挑战：
• 无伤通关成就

• 速通计时挑战

• 最少形态切换次数挑战

九、特殊机制

漏油表现

• 直升机持续从底部滴落"油"字

• 油滴在地面形成小水洼

• 水洼存在5秒后消失

• 视觉提示燃油消耗，但无实际掉血效果

螺旋桨状态

正常旋转：顺时针匀速旋转
被击中：停转0.5秒
阶段转换：加速旋转
濒死：旋转不稳定，时快时慢

机枪冷却

• 阶段A激光发射时，机枪口发热变红

• 连续攻击3次后需要冷却2秒

• 冷却期间机枪口变暗

十、隐藏机制

快速阶段转换

条件：在阶段转换前造成超额伤害
效果：跳过当前阶段剩余攻击
示例：阶段A还剩5血时一击造成10点伤害，直接进入阶段B

完美弹反

条件：在导弹即将命中的瞬间弹反
效果：导弹伤害提升至30点
视觉：导弹变成金色，尾焰更大

连续正确应对

条件：阶段D连续正确应对3轮攻击
效果：获得5秒输出时间，直升机眩晕
奖励：可造成双倍伤害

设计总结

这个直升机BOSS战设计强调：

1. 严格的形态克制：每个阶段必须使用对应角色
2. 清晰的阶段划分：四阶段对应三种能力+最终测试
3. 逐步提升的难度：从单一攻击到混合快速攻击
4. 充分的教学引导：每个阶段首次进入都有详细提示
5. 丰富的视觉反馈：明确的预警、命中特效、阶段提示
6. 深度的策略性：需要掌握时机、切换节奏、反击技巧
7. 剧情融合：与黑曼巴的复仇故事紧密结合

玩家在这场战斗中需要：
• 熟练掌握三个角色的核心能力

• 快速判断形势并切换形态

• 精确把握反击时机

• 保持冷静应对越来越快的攻击节奏

直升机不仅是黑曼巴的宿敌，也是玩家能力的综合测试，为最终的多形态协同作战打下基础。
